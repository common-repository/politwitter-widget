Canadian political twitter widget.

== Description ==

Canadian political twitter widget. Display the latest political tweets from http://politwitter.ca/ with options to only show tweets from a specific party, province or MPs only.

== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Configure the widget using the 'Widgets' menu in WordPress
